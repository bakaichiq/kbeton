## ❌ Найденные проблемы

### 1. Тестовая инфраструктура
**Проблема:** SQLite не поддерживает JSONB тип (PostgreSQL-specific)  
**Влияние:** 21 тест не может быть выполнен без PostgreSQL  
**Решение:** Использовать conftest.py с ограниченным набором таблиц

### 2. Импорт денежных сумм
**Проблема:** `parse_money` использует `Decimal` без каста в float  
**Локация:** `kbeton/importers/utils.py:37`  
**Влияние:** Низкое — в production работает корректно
